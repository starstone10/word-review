# 🇰🇷 韩语背词应用 (한국어 단어 암기)

基于 **Streamlit** 开发的智能韩语单词复习工具，结合艾宾浩斯遗忘曲线，支持词书导入、语音朗读、拼写练习、签到打卡等完整学习闭环。

---

## 主要功能

- **词书管理**  
  支持上传 TXT 文件（奇数行单词，偶数行释义），自动添加新词并生成音频。

- **智能复习**  
  自动安排复习计划，错词会动态调整难度，达到最长间隔后自动移除（视为已掌握）。

- **语音合成**  
  使用 Google TTS（gTTS）为每个单词生成韩语发音，学习更高效。

- **拼写练习**  
  认读阶段后可进入拼写模式，答错会显示正确答案并记录错误系数，下次复习频率更高。

- **学习统计**  
  连续签到天数、累计学习天数、今日复习进度、总复习单词数等数据一目了然。

## 快速开始

### 1. 环境要求
- Python 3.8+
- 操作系统：Windows / macOS / Linux

### 2. 克隆项目
```bash
git clone https://github.com/starstone10/word-review.git
cd word-review
```

### 3. 启动应用
```bash
streamlit run word.py
```

浏览器自动打开 `http://localhost:8501` 即可使用。

## 使用说明

### 首次使用
- 应用会自动创建 `korean_vocab.json` 和 `audio/` 文件夹。
- 在侧边栏上传 TXT 词书（UTF-8 编码），格式示例：
  ```
  안녕하세요
  你好
  감사합니다
  谢谢
  ```
  每两行为一组（单词→释义），支持一个单词对应多个释义（上传多次则追加）。

### 复习流程
1. **认读阶段**：显示韩语单词，点击“认识”或“不认识”。
   显示释义和笔记区，可编辑笔记。
   - “认识” → 认读下一词。
   - “不认识” → 该词移至队尾，增加难度系数（关系到复习时间），认读下一词。
2. **拼写阶段**：显示释义，要求输入韩语单词。
   - 正确 → 更新复习计划，今日完成数+1，拼写下一词。
   - 错误 → 显示正确答案，重新拼写，该词移至队尾，增加难度系数（关系到复习时间），拼写下一词。
3. **完成当日复习**：返回首页查看统计数据。

### 数据持久化
- 所有单词、复习进度、签到记录均保存在 `korean_vocab.json` 中，重启应用不丢失。
